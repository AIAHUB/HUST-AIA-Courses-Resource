# HUST-AIA-Courses-Resource
> 这是一个收集华中科技大学（HUST）人工智能与自动化学院（AIA）课程资源的repo；
>
> 课程包括了HUST开设的公共必修课，部分公共选修课，以及学院开设的专业选修课程；
>
> 资料内容包括课程电子版教材教辅，课件ppt，考试试卷，课程作业，学习心得等；
>
> 目的是为学弟学妹提供参考，帮助大家顺利通过考试！
>
